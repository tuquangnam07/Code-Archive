## 📋 Tổng quan

**Code Archive** là nơi lưu giữ toàn bộ dự án, tool, script bạn từng viết. Hoạt động như một cuốn nhật ký lập trình, lưu lại thời gian, trạng thái và câu chuyện đằng sau mỗi dòng code.

### ✨ Tính năng

- 📊 **Dashboard**: Thống kê tổng quan, project mới nhất
- 📚 **Danh sách**: Card project với đầy đủ thông tin
- 🔍 **Tìm kiếm & Lọc**: Theo tên, ngôn ngữ, trạng thái, năm
- 📄 **Chi tiết**: Ảnh preview, ghi chú, link GitHub/Download
- 🌙 **Dark mode**: Phong cách GitHub, responsive
- 🎨 **Trạng thái**: Working (xanh), Broken (đỏ), In Progress (vàng), Deprecated (xám)
